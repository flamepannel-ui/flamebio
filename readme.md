# @trgdm Terminal Portfolio

A terminal-themed portfolio website with a full-stack architecture.

## Tech Stack

### Backend
- FastAPI
- MongoDB (via Motor)
- Python 3.12+

### Frontend
- React 19
- Tailwind CSS
- Framer Motion
- React Router

## Live Site
https://flame.cafe

## Deployment
This project is configured for deployment on Railway.

## Environment Variables

### Backend
- `MONGO_URL`: MongoDB connection string
- `DB_NAME`: Database name
- `CORS_ORIGINS`: Comma-separated list of allowed origins

### Frontend
- `REACT_APP_BACKEND_URL`: Backend API URL
